## 使用技術：

### 1.react SAP 應用

react-hook 的使用，包含 react-router-dom 不重整換頁(SPA), useContext 的資料管理，dispatch 的更改動作

### 2.ant design 圖表呈現

使用 ant design 的 pie chart 做統計資料

### 3.styled-component

css-in-js 也利用 props 傳一些共通使用的參數

### 4.localStorage

使用 localStorage 存放每筆資料，更新後也能存在
